# Shelf Music Library

Shelf Music Library is a Laravel web app where users can upload songs (MP3/WAV), browse the library, and manage playlists (cart-like feature).

## Features
- Upload songs with optional cover image
- Library browsing + filtering/search
- Authentication (users/roles)
- Playlists (add/remove songs)

## Tech Stack
- Laravel (PHP)
- MySQL
- Blade + JavaScript
- Vite / npm

## Requirements
- PHP (with extensions required by Laravel)
- Composer
- Node.js + npm
- MySQL
